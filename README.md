# 🌄 West Virginia Economic Analysis – Data Pipeline & Dashboard

## 📌 Project Overview

This project is an upgraded, end-to-end version of a previously completed team-based economic analysis of West Virginia.

The original project focused on workforce dynamics, industry structure, growth patterns, and business activity across the state.
This version expands that work into a production-style data project with a focus on data engineering, reproducibility, and interactive analytics.

## 🎯 Objectives

* Rebuild the original analysis using a structured data pipeline
* Store and manage data in a PostgreSQL database
* Perform SQL-based analysis
* Develop an interactive dashboard for data exploration
* Demonstrate real-world data engineering and analytics workflows

## 🏗️ Project Components

* Python data pipeline (ETL: Extract, Transform, Load)
* PostgreSQL database integration
* SQL queries for analysis
* Streamlit dashboard application

## 📈 Project Evolution

This project builds on insights and datasets from the original West Virginia Economic Trends analysis and enhances them with:

* Modular, production-style code
* Automated data processing
* Scalable data storage
* Improved data accessibility through interactive visualization
