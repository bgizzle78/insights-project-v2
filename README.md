# 🌄 West Virginia Economic Analysis – Data Pipeline & Dashboard

## 📌 Project Overview
This project analyzes workforce dynamics, industry structure, and economic trends in West Virginia.

It is built as an end-to-end data project including:
- Python data pipeline (ETL)
- PostgreSQL database
- SQL analysis
- Interactive dashboard (Streamlit)
